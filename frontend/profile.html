1️⃣

สร้าง Telegram Bot

คุยกับ @BotFather ใน Telegram แล้วส่ง /newbot

บันทึก Bot Token ที่ได้ (Admin จะนำไปตั้งค่าระบบ)

2️⃣

หา Chat ID ของคุณ

ไปที่ @userinfobot แล้วส่ง /start

Bot จะตอบกลับด้วย Chat ID (ตัวเลข) ของคุณ — คัดลอกมาใส่ด้านล่าง

3️⃣

⚠️ สำคัญ: เริ่มคุยกับ Bot ก่อน!

คุณต้องส่ง /start ไปที่ Bot ก่อน — ไม่งั้นระบบจะส่งข้อความหาไม่เจอ
สอบถาม Admin เพื่อขอ username ของ Bot

4️⃣

บันทึก Chat ID

ต้องเป็นตัวเลขเท่านั้น (หาได้จาก @userinfobot)

💚 เชื่อมต่อ LINE (รับแจ้งเตือนเมื่อซ่อมเสร็จ)

✅

เชื่อมต่อ LINE แล้ว

คุณจะได้รับข้อความ LINE ทันทีเมื่อครุภัณฑ์ของคุณซ่อมเสร็จ

1️⃣

เพิ่มเพื่อน LINE Official Account

สแกน QR Code หรือค้นหา LINE OA ของระบบแจ้งซ่อม แล้วกดเพิ่มเพื่อน (สอบถาม Admin หากยังไม่มี QR Code)

2️⃣

ขอรหัสเชื่อมต่อ 6 หลัก

รหัสมีอายุ 10 นาที
